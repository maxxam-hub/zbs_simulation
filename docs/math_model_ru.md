# Математическая модель расчета ЗБС (текущая реализация)

## 1. Назначение модели

Модель предназначена для многовариантного сравнения сценариев ЗБС по дебиту и давлению при фиксированном забойном давлении на пятке.

Целевой выход:
- прогнозный дебит `q`;
- среднее давление по горизонтальному участку;
- распределение давления вдоль ствола;
- прирост/падение относительно вертикального эталона.

## 2. Принятые допущения

- изотермический режим (`T = const`);
- «сухой газ» на уровне MVP;
- стационарная постановка для каждого сценария;
- расчет в SI;
- сравнение сценариев при заданном `Pwf_heel`.

## 3. Базовое уравнение притока

Для каждого сценария решается двучленное уравнение по квадратам давления:

`Pпл^2 - Pз^2 = A*q + B*q^2`

где `A` и `B` формируются выбранной методикой притока.

Физически допустимое решение: `q >= 0`.

## 4. Режимы по забойному давлению

### 4.1 `pwf_const`

- давление по длине ГС принимается постоянным;
- `q` считается без итерационной увязки с гидравликой ствола.

### 4.2 `pwf_variable_iterative`

- учитывается изменение давления по длине ГС;
- на каждой итерации пересчитываются стволовые потери (НКТ/затруб), среднее давление и коэффициенты `A/B`;
- итерации завершаются по критерию сходимости дебита.

## 5. Блок PVT

### 5.1 Псевдокритические параметры

Поддерживаемые методы:
- `composition_linear` — аддитивный расчет по компонентам (аналог II.12-II.13);
- `composition_precise` — уточненный расчет по компонентному составу (аналог II.14);
- `specific_gravity_sutton` — расчет по относительной плотности газа (fallback).

### 5.2 Z-фактор

Поддерживаемые методы:
- `dranchuk_abou_kassem` (основной по умолчанию);
- `aliev_zotov_two_parameter` (аналитический метод, аналог II.21);
- `papay` (упрощенный метод).

### 5.3 Вязкость и плотность

`μ(P,T)` и `ρ(P,T,Z)` рассчитываются в `reservoir.py` и используются в притоке и гидравлике ствола.

## 6. Методики расчета коэффициентов A/B

### 6.1 `legacy_empirical`

Исходная эмпирическая постановка проекта.
Геометрический множитель сопротивления учитывает:
- длину ГС;
- анизотропию;
- асимметрию положения ствола по толщине пласта;
- радиус кривизны;
- `skin`.

### 6.2 `aliev_2015_anisotropic`

Инженерная анизотропная постановка для ГС:
- явный учет `ae = sqrt(kv/kh)`;
- геометрический член с анизотропной поправкой;
- поправки на положение входа и радиус кривизны;
- далее формирование `A` и `B` в общей форме двучленного уравнения.

Важно:
- это адаптация под расчетный контур проекта;
- не буквальная «1 в 1» реализация всех формул 3.17-3.19.

## 7. Геометрия профиля ГС

Поддерживаются профили:
- `flat` — горизонтальный;
- `ascending` — линейный подъем;
- `descending` — линейный спуск;
- `stepped` — ступенчатый профиль (вручную задаются длины ступеней и перепады высот).

Для `stepped` шаблон длин автоматически масштабируется под фактическую длину ГС сценария.

## 8. Гидравлика ствола (Completion)

Ствол делится на сегменты:
- участок потока в НКТ;
- участок потока в затрубном пространстве (если башмак не у забоя).

Для каждого сегмента учитываются:
- потери на трение (Darcy-Weisbach, коэффициент трения через явную аппроксимацию Colebrook/Swamee-Jain);
- гравитационная составляющая через изменение высоты профиля.

Сумма сегментных потерь задает `ΔP_wellbore`.

## 9. Алгоритм расчета сценария

1. Задать сценарий (`L`, `Rкр`, профиль, башмак НКТ, `ae`).
2. Получить начальные `A/B` и стартовый `q`.
3. При `pwf_variable_iterative` пересчитывать потери по стволу и среднее давление.
4. Пересчитывать `A/B` и новый `q` до сходимости.
5. Сохранить `q`, давления, `A`, `B`, число итераций и прирост к вертикальному эталону.

## 10. Многовариантный анализ

Перебор выполняется по сетке:
- `Lг`;
- `Rкр`;
- профиль траектории;
- глубина башмака НКТ;
- `ae`.

Результаты сохраняются в `outputs/scenario_results.csv` и `outputs/scenario_results_ru.csv`.

## 11. Использованные источники (по текущей модели)

- «2.3 АН 2021», стр. 50: двучленная форма уравнения притока.
- Алиев (2015), гл. 3: постановка расчета производительности ГС и учет анизотропии (как методическая база для режима `aliev_2015_anisotropic`).
- «Инструкция по комплексному исследованию…» (1980), гл. II:
  - II.12-II.14 — псевдокритические параметры;
  - II.21 — аналитический расчет `Z`.

## 12. Ограничения текущей версии

- нет полной покомпонентной EOS-модели уровня compositional simulator;
- нет калибровки по историческим точкам (`Q`, `Pwf`, дата, состояние НКТ);
- для `aliev_2015_anisotropic` используется инженерная адаптация, а не полный набор формул главы 3 в закрытой форме.

Эти ограничения не мешают сценарианому сравнению вариантов ЗБС в рамках курсовой и этапа MVP.
