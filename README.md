# Симулятор ЗБС (скважина 2205)

Python-проект для многовариантного расчета дебита и давления в боковом горизонтальном стволе (ЗБС) с учетом:
- свойств газа `Z(P,T)` и `μ(P,T)`;
- анизотропии пласта `ae = sqrt(kv/kh)`;
- геометрии траектории (плоский, восходящий, нисходящий, ступенчатый профиль);
- гидравлики НКТ и затрубного пространства;
- сравнения с вертикальным эталоном при фиксированном забойном давлении на пятке.

## 1. Быстрый запуск

```bash
python -m pip install -r requirements.txt
python run_simulation.py
```

После запуска результаты будут в папке `outputs/`.

## 2. Что считает модель

Базовое уравнение притока:

`Pпл^2 - Pз^2 = A*q + B*q^2`

Где:
- `q` — дебит газа (ст. м3/сут);
- `A` — линейное фильтрационное сопротивление;
- `B` — нелинейное (инерционное) сопротивление.

В проекте есть два режима по забойному давлению:
- `pwf_const` — давление по ГС принимается постоянным;
- `pwf_variable_iterative` — итерационная увязка притока и стволовых потерь давления.

## 3. Текущие методы (переключаются в конфиге)

### 3.1 Коэффициенты притока A/B
- `legacy_empirical` — исходная эмпирическая схема проекта.
- `aliev_2015_anisotropic` — инженерная анизотропная постановка для ГС (адаптация под подход Алиева, с явным учетом `ae`).

### 3.2 Псевдокритические параметры
- `composition_linear` — аддитивный расчет по компонентному составу.
- `composition_precise` — уточненный расчет по составу (K/J).
- `specific_gravity_sutton` — fallback по относительной плотности газа.

### 3.3 Z-фактор
- `dranchuk_abou_kassem` — основной рабочий метод (по умолчанию).
- `aliev_zotov_two_parameter` — аналитический метод по формуле (II.21) из Инструкции 1980.
- `papay` — упрощенная быстрая аппроксимация.

## 4. Где менять параметры (пошагово)

Все основные настройки задаются в `src/zbs_sim/models.py`.

1. Откройте `ReservoirParams`.
2. Установите метод притока: `inflow_method`.
3. Установите режим по `Pз`: `pwf_mode`.
4. Установите методы PVT: `ppc_method` и `z_method`.
5. Проверьте пластовые параметры: `p_res_mpa`, `t_res_c`, `k_mD`, `h_m`, `ae_default`.
6. Задайте компонентный состав в `gas_composition_mol_frac`.
7. Откройте `WellboreParams`.
8. При необходимости задайте угол линейных профилей: `ascending_angle_deg`.
9. Для ступенчатого профиля задайте:
   - `stepped_segment_lengths_m` — длины ступеней;
   - `stepped_step_heights_m` — перепады высот между соседними ступенями.
10. Откройте `SweepParams`.
11. Задайте сетку перебора: `lateral_lengths_m`, `curvature_radii_m`, `profiles`, `anisotropy_values`, `shoe_step_m`.
12. Запустите `python run_simulation.py`.

## 5. Профили траектории ГС

Поддерживаются профили:
- `flat` — плоско-горизонтальный;
- `ascending` — линейно восходящий;
- `descending` — линейно нисходящий;
- `stepped` — ступенчатый профиль.

Для `stepped` шаблон длин автоматически масштабируется под фактическую длину ГС в каждом сценарии.

## 8. Структура проекта

- `run_simulation.py` — запуск расчета, сохранение таблиц и графиков.
- `src/zbs_sim/models.py` — все входные параметры и конфигурация.
- `src/zbs_sim/reservoir.py` — физика газа и PVT-корреляции.
- `src/zbs_sim/inflow.py` — расчет коэффициентов `A/B` по выбранной методике.
- `src/zbs_sim/geometry.py` — геометрия ствола и профиль траектории.
- `src/zbs_sim/completion.py` — потери давления в НКТ и затрубе.
- `src/zbs_sim/flow_engine.py` — решатель сценария (итерационная увязка).
- `src/zbs_sim/sensitivity.py` — перебор сценариев и сбор итоговой таблицы.
- `src/zbs_sim/visualization.py` — построение графиков.
- `docs/math_model_ru.md` — формальное описание матмодели.

## 9. Важно по интерпретации

- Метод `aliev_2015_anisotropic` сейчас реализован как инженерная адаптация под задачу курсового проекта, а не буквальная подстановка всех формул главы 3 «1 в 1».
- Для сравнений «до/после» сохраняйте baseline-таблицу отдельным файлом перед правками.
