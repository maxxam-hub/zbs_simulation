# Многовариантный аналитический симулятор ЗБС (MVP)

Минимальный Python-проект для сравнения вариантов бокового горизонтального ствола по скважине 2205.

## Текущая постановка модели

- Изотермический режим (`T = const`).
- Допущение «сухой газ».
- Стационарный расчет для каждого сценария.
- Фиксированное забойное давление на пятке (`Pwf_heel`) для сравнения с вертикальным эталоном.
- Уравнение притока:
  - `P_res^2 - P_sf^2 = A*q + B*q^2`
- Перебор сценариев:
  - Длина горизонтального участка (`100..1500 м`, шаг `100 м`).
  - Радиус кривизны (`8, 40, 120 м`).
  - Тип профиля (`flat`, `ascending`).
  - Глубина башмака НКТ от пятки до забоя (`0..L`, шаг `100 м`).
  - Анизотропия `ae` (`0.1, 0.2, 0.3`).

## Зафиксированные входные параметры

- `P_res = 9.5 МПа`
- `T_res = 46 C`
- `k = 345 мД`
- `h = 40 м`
- Относительная плотность газа `gamma_g = 0.693`
- `CH4 = 96.23%`, `C2H6 = 3.02%` (информационно для MVP)
- `КГФ = 0.85 г/м3` (в MVP оставлено допущение сухого газа)
- Номинальный диаметр НКТ `114 мм`
- Текущая глубина спуска НКТ `1567.1 м` (в переборе башмак перемещается по ГС)

## Текущие зависимости PVT

- Псевдокритические параметры:
  - при наличии состава: правило Кея (Инструкция 1980, гл. II.3, формулы II.12-II.13),
  - fallback: Sutton по относительной плотности.
- `Z(P,T)` по умолчанию: Dranchuk-Abou-Kassem (`z_method = "dak"`).
- Альтернатива: `Papay` (`z_method = "papay"` в `ReservoirParams`).

## Быстрый запуск

```bash
python run_simulation.py
```

Опционально (чтобы получать PNG-графики, а не только CSV-данные для них):

```bash
python -m pip install -r requirements.txt
```

## Структура модулей

- `src/zbs_sim/reservoir.py`: физика газа (`Z`, `mu`, `rho`).
- `src/zbs_sim/geometry.py`: геометрия ствола и геометрическое сопротивление.
- `src/zbs_sim/completion.py`: сегментация НКТ/затруб и стволовые потери давления.
- `src/zbs_sim/flow_engine.py`: итерационная связка притока и гидравлики.
- `src/zbs_sim/sensitivity.py`: многовариантный перебор и таблица результатов.
- `src/zbs_sim/visualization.py`: построение графиков и экспорт данных графиков.

## Что улучшать далее

- При необходимости заменить Papay на более детальную корреляцию `Z(P,T)`.
- Подставить паспортные внутренние диаметры НКТ/ЭК для повышения точности.
- Добавить калибровку по фактическим точкам (`Q`, `Pwf`, дата, состояние НКТ).
